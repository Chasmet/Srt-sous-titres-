Mode conseillé : aperçu propre

L’aperçu utilise ton MP4 original, donc le rendu est propre. Si tu as déjà un SRT, colle-le directement dans la zone SRT puis appuie sur “Lire le SRT collé”.

2. Génère ou colle le SRT
3. Vérifie l’aperçu
4. Crée 3 vidéos à recoller dans CapCut
